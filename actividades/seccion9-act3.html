el elemento div permite agrupar un conjuntio de elementos juntos en un cuadro de nivel de bloque. por ejemplo, puede crear un elemento div; para que contenga todos los elementos del encqabezado del sitio(el logotipo y la naveagacion), o puede crear un elemnto div para que contenga informacion en bloques

en un navegador, el contenido del elemento div comenzara en una nueva linea, pero aparte de esto no habra ninguna diferencia en la presentacion de la pagina sin embargo, el uso de un atributo ID o class en el espacio debe ocupar el elemento div en la pantalla y cambiar la aparencia de todos los elementos contenidos en el. tambien puede facilitar el seguimiento del codigo si ha utilizado elementos div para contener cada seccion de la pagina.

dado que puede haber varios otrto elemntos dentro de un elemento div puede ser util agregar un comentario despues de la etiqueta div de cierre. esto le permite ver claramente a que etiqueta de apertuira se supone que corresponde.
