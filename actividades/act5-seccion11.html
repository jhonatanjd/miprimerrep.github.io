css selectors

hay muchos tipos diferentes de selectors css que le permiten orientar las reglas a elementos especificos en un documento html.

los selectores css distinguen entre mayusculas y minusculas , por lo que deben coincidir exactamnete con los nombres de elemento y los valores de atributo
