background color - color de fondo

si no se especifica un color de fondo, el fondo es trasparente.

css trata cada elemento html como un cuadro y la propiedad establece el color del fondo para el cuadro. se puede especificar un color de fondo de mlas mismas tres formas en que puede especificar colores de primer plano valores rgb, codigo hexadecimal y nombre de colores. tambien hemos utilizado la propiedad padding para separar el texto de los bordes de los cuadros. esto hace que sea mas facil de leer.
