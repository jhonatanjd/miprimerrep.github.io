html5 introdujo el elemento audio para incluir aechivops de audios en sus paginas . al igual que con el video html5 los navegadores esperan diferentes formatos para el audio. el elemento audio tiene un numero de atributos que le permiten controlar la reproduccion de audio:

src: este atributo especifica la ruta de acceso al archivo de audio

controls: este atributo indica si el rproductor debe mostrar los controles. si no se utiliza este atributo, no se mostraran los controles mediante javaScript

autoplay lan presencia de este atributo indica que el audio debe empezar reproducirse automaticamente (se considera una mejor practica permitir que los visitantes elijan reproducir audio.)

loop: este atriobuto especifica que la pista de audio debe reproducirse de nuevo una vez que haya finalizado
